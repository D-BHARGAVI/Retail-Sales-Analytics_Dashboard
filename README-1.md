# Retail Sales Intelligence Dashboard

=>Project Overview
This project focuses on analyzing retail sales data using SQL, Python, and Tableau to generate business insights and interactive visualizations.

The project includes data cleaning, SQL analysis, exploratory data analysis, and dashboard development using real-world retail sales data.


=> Tools & Technologies Used
- Python
- Pandas
- NumPy
- MySQL
- Tableau Public
- Jupyter Notebook


=> Features
- Sales Performance Analysis
- Profit Analysis
- Region-wise Sales Insights
- Category-wise Sales Comparison
- Monthly Sales Trend Analysis
- State-wise Sales Visualization
- Interactive Tableau Dashboard


=> Dashboard Components
- KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders

- Charts & Visualizations
  - Monthly Trends
  - Region Sales
  - Category Sales
  - State Sales Map
